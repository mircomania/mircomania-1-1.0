# Stack Tecnológico

## Frontend

- Next.js (App Router)
- React
- TypeScript
- CSS Modules

## Backend

- Next.js Route Handlers (cuando corresponda)
- Node.js

## Base de datos

- Supabase
    - PostgreSQL
    - Storage
    - Row Level Security

## Hosting

- Vercel
- Hosting Serverless

## Imágenes

- Next Image

## SEO

- Metadata API
- Server Components
- SSR

## Arquitectura

- Mobile First
- Componentes reutilizables
- Separación por responsabilidades

## Librerías

- Utilizar librerías solo cuando realmente aporten valor.
- Preferir las APIs nativas del navegador antes de instalar dependencias.
